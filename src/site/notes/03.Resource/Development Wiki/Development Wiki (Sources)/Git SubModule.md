> 하나의 저장소에 다른 저장소를 포함한다

- 메인 리포지트리와 서브모듈 리포지트리 관계
    - 메인 리포지트리는 복수개의 서브 모듈을 가질 수 있다
        - 메인 1 : 서브 N
    - 메인 리포지트리에서 서브모듈 리포지트리를 수정할 수는 없다
    - 서브모듈 리포지트리가 변경되면 메인 리포지트리에서 변경사항을 업데이트 할 수 있다
- 왜 쓰는가?
    - 여러 프로젝트에서 사용할 패키지 모음들을 쉽게 모듈화하여 관리할 수 있음
