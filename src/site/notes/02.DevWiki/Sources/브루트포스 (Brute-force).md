# 브루트포스 (Brute-force)

> **가능한 모든 경우의 수를 전부 탐색**하여 문제를 해결하는 알고리즘
* O(n)
- 가장 **간단하고 직관적**인 문제 해결 방법.
- 해를 반드시 찾을 수 있다는 장점이 있지만 **비효율적**.
### 사용처
- 문제의 크기가 작아 성능이 크게 문제가 되지 않을 때.
- 다른 효율적인 알고리즘을 찾기 전에 문제의 답을 확인하고 싶을 때.
- 암호 해독시 무차별 대입.
