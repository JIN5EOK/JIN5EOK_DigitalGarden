- **저장 메모리 차이**
    - **값**
        - **스택** 메모리에 **값 저장**
    - **참조**
        - **스택** 메모리에 **주소 저장**
        - **힙** 메모리에 **값 저장**
- **메모리 해제 과정**
    - **값**
        - **사용 종료후** 메모리 해제
    - **참조**
        - 언어에 따라 **프로그래머가 해제** 또는 **가비지 컬렉터**가 해제
